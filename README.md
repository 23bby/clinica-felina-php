# Clinica Felina PHP
Dashboard gestionale per una clinica veterinaria realizzata in PHP e MySQL.

## Caratteristiche
- Gestione appuntamenti
- Inserimento dati gatti e proprietari
- Statistiche cliniche
- Interfaccia basata su AdminLTE

## Installazione
1. Importa il file `.sql` nel tuo database locale.
2. Configura i parametri di connessione in PHP.
3. Avvia tramite XAMPP.


# Feline Clinic Dashboard (PHP)
A management dashboard for a veterinary clinic developed using PHP and MySQL.

## Features
- **Appointment Management**: Organize and track veterinary visits.
- **Data Entry**: Easily manage records for cats and their owners.
- **Clinical Statistics**: View data and analytics for the clinic.
- **Modern UI**: Interface built on the AdminLTE framework.

## Installation
1. Import the provided `.sql` file into your local database.
2. Configure the connection parameters in the PHP configuration files.
3. Run the project using XAMPP or any local PHP server.
